# Badges BCH x IA

Conjunto de insignias digitales para reconocer contribuciones al proyecto BCH x IA.

## Badges disponibles

| Badge | Descripción | Cómo obtener |
|-------|-------------|---------------|
| **Pionero BCH‑IA** | Primer contribuidor que abre un PR exitoso | Automática al mergear primer PR |
| **Integrador BCH** | Implementa `bch-pay-client` en su proyecto | Enlace al repo + mention en issues |
| **Escritor Técnico** | Mejora la documentación | +5 PRs en docs/ |
| **Community Hero** | Ayuda en Discord/Reddit/foros | Nominación por la comunidad |

Más badges se añadirán según crezca el proyecto.

## Uso

Los badges son imágenes SVG/PNG que puedes usar en:
- READMEs de GitHub
- Perfiles de Twitter/LinkedIn
- Sitios web personales

Ejemplo en markdown:
```markdown
![Pionero BCH‑IA](https://raw.githubusercontent.com/y42bvf6695-gif/bch-ia-badges/main/badges/pionero-bch-ia.svg)
```

## Generación automática (futuro)

Se planea automatizar la entrega mediante GitHub Actions que:
1. Detecten PRs mergerados con ciertas etiquetas
2. Actualicen el perfil del usuario con el badge correspondiente
3. Envíen notificación por Discord
