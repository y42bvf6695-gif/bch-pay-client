- Creado `examples/send_mainnet.py` para envíos manuales de BCH en mainnet via Paytaca

#### 3. QA Agent con envío real opcional
- Añadida opción `--real-send` al QA agent
- Solo funciona con `--backend paytaca` y `--network mainnet`
- Requiere `--address` y `--amount`
- Pide confirmación escribiendo "REAL" (seguridad extra)
- Use case: integration testing con wallets de prueba

## Comandos útiles

### Ejecutar QA suite (demo)
```bash
.venv/bin/python examples/agent_venture_qa.py --backend demo --network testnet --report markdown
```

### Ejecutar QA con envío real (⚠️ solo testing wallets)
```bash
.venv/bin/python examples/agent_venture_qa.py \
  --backend paytaca \
  --network mainnet \
  --real-send \
  --address bitcoincash:qz8983kqe0pwccwg3urj462mlzrm02q4vc3z22j9w4 \
  --amount 0.00242019
```

### Enviar BCH manualmente
```bash
python examples/send_mainnet.py --amount 0.00242019 --address bitcoincash:qz8983kqe0pwccwg3urj462mlzrm02q4vc3z22j9w4
```
